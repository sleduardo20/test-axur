## Instalação

Siga as seguintes etapas no seu prompt de comando:

```
git clone git@github.com:sleduardo20/test-axur.git
```
```
npm install
# ou
yarn
```

```
npm run dev
# ou
yarn dev
```

## Testes
```
npm run test
# ou
yarn test
```

## Produção
```
npm run build
# ou
yarn build
```

```
npm run start
# ou
yarn start
```
